Código em C

        
#include <stdio.h>

const int senha = 2002;

int main(void)
{
	int num;
	int exit = 0;
	while (!exit)
	{
		scanf("%d", &num);
		if (senha == num)
		{
			printf("Acesso Permitido\n");
			exit = 1;
		}
		else
		{
			printf("Senha Invalida\n");
		}
	}
}

Esse programa em C serve para verificar se a senha digitada está correta. A senha correta é 2002. O programa fica pedindo para o usuário digitar uma senha (com scanf) até que ele acerte. Se a senha digitada estiver errada, ele mostra "Senha Invalida". Quando o usuário digita a senha certa (2002), ele mostra "Acesso Permitido" e o programa termina.
