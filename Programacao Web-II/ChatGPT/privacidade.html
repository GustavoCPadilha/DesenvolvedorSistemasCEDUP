Política de Privacidade - ChatGPT para Pobres

Última atualização: 02/09/2025

Oi! Aqui a gente explica rapidinho como lidamos com seus dados e sua privacidade no ChatGPT para Pobres.

  1. Sobre o que é isso?

    O ChatGPT para Pobres não é uma inteligência artificial de verdade, é só uma ferramenta simples para ajudar você a trocar umas ideias.

  2. Dados que coletamos

    A gente não coleta informações pessoais suas. Se você quiser compartilhar algo no chat, é por sua conta e risco.

  3. Como usamos os dados

    Se você mandar alguma informação, ela será usada só para responder suas mensagens durante a conversa.

  4. Compartilhamento de dados

    Não compartilhamos suas informações com ninguém, prometo.

  5. Segurança dos seus dados

    A gente tenta proteger o que você manda, mas por ser um serviço simples, não dá pra garantir 100% de segurança.

  6. Seus direitos

    Você pode escolher não mandar dados pessoais e, se quiser, pode pedir para apagar informações enviadas.
