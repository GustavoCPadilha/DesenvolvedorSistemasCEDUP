Que significa semântica? Por que isso é tão importante em HTML5?

A semântica é extremamente importante para trazer significado ao conteúdo do seu site. 😀 Isso é importante para separar A aprte de estilização do site do contéudo mas principalmente, para que o algorítmo dos navegadores consiga entender o propósito do seu site. 😆
Por exemplo: uma palavra em negrito não é apenas uma palavra em negrito para o HTML Semântico, o negrito é uma palavra em destaque, para que quando algum usuário pesquise essa mesma palavra, o navegador recomende seu site. 🫢
