As Três Formas de Usar CSS

CSS Inline

  • Estilos escritos direto no elemento HTML, usando style.
  • Vantagens: rápido para mudar um só item.
  • Desvantagens: difícil de manter e não reutiliza.

CSS Interno

  • Estilos dentro da tag <style> no arquivo HTML.
  • Vantagens: centraliza o estilo da página.
  • Desvantagens: só funciona naquela página e pode deixar o arquivo pesado.

CSS Externo

  • Estilos em arquivo separado, ligado pelo <link>.
  • Vantagens: reutiliza em várias páginas, fácil de organizar.
  • Desvantagens: precisa carregar um arquivo extra, pode atrasar um pouco o estilo.
